Tree preservation orders

Trees make places more attractive, provide valuable habitats for wildlife, improve the air we breathe and help to conserve energy. Sometimes they need long-term protection from harm.

Local planning authorities have wide powers to make tree preservation orders to protect trees of significant value to communities. These written orders, in general, make it an offence to prune, fell, damage or destroy the trees they cover without local planning authority consent.

The tree preservation order system is governed by the Town and Country Planning (Tree Preservation) (England) Regulations 2012. Following public consultation, these regulations came into force on 6 April 2012. They replace three sets of regulations and make the system simpler for local authorities to administer and easier and fairer for tree owners.

There is a summary of the main changes brought about by the new regulations in Main changes to the tree preservation order system in England from 6 April 2012.

If you would like to check whether a tree is protected, ask for a tree to be protected or apply for consent for work to a protected tree please contact the local planning authority. Please also see Protected trees: a guide to tree preservation procedures. This leaflet answers common questions about the tree preservation order system and tree protection in conservation areas in England.
