Accessibility and help using the internet

We’re trying to make this site as accessible and usable as we can for everyone.

Your feedback will really help us to do that, so if you spot any problems please visit the feedback page and let us know. And, if you’re interested, this blog post will tell you more about the accessibility testing we’ve been doing.
