Service File an annual return with Companies House

If you run a company you must send an annual return to Companies House. This includes details of directors, registered office address, shares and shareholders.

What you need to know

Start now on the Companies House website

  • Before you start
  • Other ways to apply

It costs £13 to file your annual return online.

You can pay by debit/credit card or Paypal.

You’ll need a Companies House password and authentication code.

If you don’t have these, you can register online. You’ll get the password by email. The authentication code will be posted to your company’s registered office address.

You must file your annual return within 28 days of the anniversary of the company being set up (incorporated).

If you don’t file your annual return on time, Companies House can dissolve your company and prosecute you.

You can fill in form AR01 and post it to Companies House with your payment. It costs £40 to file your annual return on paper.
