Set up an MOT test station, a guide from GOV.UK

Notes

Part 1: Overview

If you want to set up an MOT test station, you’ll have to meet a number of requirements.

You’ll need:

  • Authorised Examiner (AE) status
  • approved MOT equipment
  • suitable premises

The equipment and premises you’ll need will depend on the vehicle classes you want to test.

Part 2: Your application

You must have Authorised Examiner (AE) status to set up an MOT test station.

An AE can be an individual, partnership or company authorised by the Vehicle and Operator Services Agency (VOSA).

The AE is responsible for making sure that:

  • the test facilities and equipment are checked and well-maintained
  • the Nominated Testers are assessed correctly and complete training and quality checks

To apply for AE status, send form VT01 to VOSA. Use the same form if you already have AE status and want to open a test station.

Download ‘Form VT01 - application for AE status / opening an MOT test station’ (PDF, 274KB)

Send the form to your local VOSA office.

Premises and equipment

Different classes of vehicle require different test equipment and testing bay sizes. You’ll need to make sure you have at least the minimum level of acceptable equipment for each vehicle class you’re approved to test.

Download ‘Requirements for authorisation for a vehicle testing station’ (PDF, 47KB)

Part 3: Vehicle classes

When deciding which class of vehicle you’d like to test, you’ll need to consider:

  • the equipment needed to test the vehicles
  • the size and layout of your premises
  • whether the Nominated Tester is approved to test the different vehicle types you want to MOT

Requirements to test different classes

The current requirements for testing each vehicle class are in the guides below.

These are just the minimum requirements - think carefully before deciding on your layout as it may affect the size or type of vehicle you’ll be able to test.

You’ll also need to make sure that you have a Nominated Tester who’s  approved to test the type of vehicles you’re applying to test.

Part 4: Meeting ongoing standards

The Vehicle and Operator Services Agency (VOSA) will carry out regular risk-based site assessments of your station to make sure you continue to meet the standards.

This will involve checking it’s well maintained and that it offers clean and comfortable facilities, like suitable customer waiting areas.

You can find out more about the risk assessments VOSA will carry out in the guide below.
